# 浏览器跨tab通信

## 背景

最近接到一个需求，大致要求是：

1. 一个后台管理界面，一个大屏展示界面，后台管理界面能控制大屏展示界面信息；
2. 大屏界面展示界面投放到大屏中，并且唯一显示，不能多开；
3. 后台管理界面中控制大屏实时显示的信息；
4. 大屏控制自动播放音频信息。

## 需求分析

后台管理界面和大屏幕展示界面需要通信，也就是涉及不同 项目/tab 之间的通信。
